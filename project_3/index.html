02Problem

The previous iteration of the financial product result page faced significant usability challenges, including a constrained layout, ineffective filtering options, and a convoluted user flow.

Visual clutter and a weak visual hierarchy made it difficult for users to scan critical metrics like miles and fees. Furthermore, an ineffective dual CTA setup ("Apply Now" vs. "See More/Less") caused interaction friction and lowered conversion intent, directly hurting core platform monetization.

Dropdown menus hid critical choices, and drawer filters obstructed content, reducing context on mobile devices where screen real estate is premium.

04Design Strategy

Strategic Product Evolution: From User Empathy to Executable UI

The previous iteration of the financial product result page faced significant usability challenges, including a constrained layout, ineffective filtering options, and a convoluted user flow.

Visual clutter and a weak visual hierarchy made it difficult for users to scan critical metrics like miles and fees. Furthermore, an ineffective dual CTA setup ("Apply Now" vs. "See More/Less") caused interaction friction and lowered conversion intent, directly hurting core platform monetization.

Dropdown menus hid critical choices, and drawer filters obstructed content, reducing context on mobile devices where screen real estate is premium.

05Solution

Modular Design System

Introduced reusable components allowing the marketing team to seamlessly reuse result page layouts across campaign and blog pages, reducing design debt and developer QA time. The modular design also strengthened SEO by enhancing internal linking structures.

Slide-Up Drawer Interaction

Implemented a drawer using tabbed navigation to reveal details (eligibility, fees) while maintaining page context, ensuring users don't lose their place in the comparison list.

Visual Hierarchy

Made key metrics (cashback %, airmiles) bold and immediately visible, aligning with the "Understand & Compare" user stage. This layout features an Urgency Banner and a left-aligned Product Hero for immediate applications.

06Impact

Helped boost search engine rankings, outperform competitors, and drive organic traffic growth through a modular design that strengthened internal linking structures.

Enabled users to scan, compare, and apply more efficiently without getting overwhelmed by dense text, leading to a higher yield per organic visit.

The reusable component library reduced design debt significantly, enabling the marketing team to independently deploy campaign pages without engineering support.
