Case Study

Reward Redemption
Flow Redesign

01 Project Overview

Project Type
Post-Conversion Loyalty UX
Company
MoneyMax · Philippines
Timeline
Apr–May 2024
Team
Customer Success, Product Management, Backend Engineering

02 Problem

While the platform was successful at acquiring users, post-conversion user satisfaction was suffering. Users successfully earned campaign rewards, but frequently abandoned the redemption flow. Key issues included confusing navigation, lack of clear status tracking, and ambiguous fulfillment timelines.

Root Cause

Users don't just need a form — they need a guided experience that helps them decide with confidence.

High Cognitive Load

Static form — all fields at once, overwhelming users with poor hierarchy

No Decision Guidance

Manual input, unclear flow — users couldn't easily browse rewards

Mobile Friction

2-column layout with small tap areas increased input errors and fatigue

Drop-off During Completion

Low confidence, high abandonment — leading to brand damage and CS tickets

03 Research and Insights

Key Observation

The original experience created high cognitive load and low user confidence, leading to drop-offs during form completion.

User Pain Points

The form displayed all fields at once, making it overwhelming and hard to navigate. Users struggled to find key information like eligibility and product details due to poor hierarchy.

The decision-making process was unclear. Users couldn’t easily browse rewards and had to manually input campaign details without guidance. On mobile, the two-column layout reduced readability, and small tap areas increased input errors.

Large images caused excessive scrolling, while the cluttered layout made it hard to focus and compare options.

Behavioral Insights

Users prefer guided, step-by-step flows over fully exposed forms. Visual elements help them understand and choose rewards faster.

Progressive disclosure reduces overwhelm by showing details only when needed. A single-column layout and larger touch targets improve mobile usability.

Design Opportunities

Simplify the layout to reduce cognitive load and improve scalability. Break content into smaller sections and highlight key information. Add interactive elements to support browsing and selection. Replace manual input with guided steps. Optimize for mobile with a clear vertical flow and touch-friendly design.

Design Direction

The experience shifted from a static form to a guided, interactive flow. Content is now structured with progressive disclosure and modular sections.

Users can select campaigns and rewards directly instead of manual input. A single-column layout with larger touch areas improves usability.

Users don’t just need a form—they need a guided experience that helps them decide with confidence.

04 Design Strategy

Goal & Principles

Enable confident transactions with clear, reversible feedback and full fee/rate transparency. Each screen shows status and next step, separates input from output, and updates results instantly.

UX Optimization

Exchange rates update live with soft alerts for changes. If balance is insufficient, the action switches to top-up. Layout keeps a clear hierarchy, hides legal content in expandable sections, and uses timelines or checklists to confirm transactions and build trust.

Frontend & Improvements

Use reusable input components for stable formatting and accessible focus states. Highlight key data, use skeleton screens for faster perception, and keep the primary action visible on small screens.

Design Opportunities

Replace manual input with guided steps. Optimize for mobile with a clear vertical flow and touch-friendly design.

05 Solution

Clear, Guided Journey

Multi-step forms with progress indicators and prominent CTAs reduce confusion and guide users seamlessly. The experience shifted from a static form to a guided, interactive flow with content structured around progressive disclosure and modular sections.

Simplified Structure & Navigation

Single-column layout, collapsible sections, and breadcrumbs improve readability and scanning. Users can now select campaigns and rewards directly instead of manual input, reducing friction dramatically.

Tap-to-Select Campaign Browsing

Replaced manual text entry with an interactive browsing and tap-to-select model. Large touch targets and a vertical-first mobile rhythm make the entire redemption flow effortless on any device.

06 Impact

Significantly boosted reward redemption completion rates by removing friction from the multi-step flow.

Reduced inbound Customer Success inquiries as users could self-serve the redemption process with clear status tracking and guided steps.

Improved post-conversion user satisfaction, turning a pain-point into a loyalty-building moment for MoneyMax Philippines users.

Established a scalable design pattern for guided flows that could be applied across other post-purchase redemption experiences on the platform.
